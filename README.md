# skl-go  

## Project description  
* 使用beego搭建企业级的管理后台开发平台  
* 平台内嵌登录、用户、用户组、公司、角色、机构、权限、OA引擎等后台管理常用功能模块。  

## Features  
* login  
* user  
* usergroup  
* company    
* role    
* orginazation    
* priviledage  
* lang    
* enumeration    
* oa engineer  

## Environment Support  
* beego 1.9.0  
* golang go1.9.2  

## Installation  
* install golang by yourself  
* install beego by yourself  
* go get github.com/mikezss/skl-go   


## Usage  
* cd %goproject%/github.com/mikezss/skl-go  
* go build main.go  
* main.exe (windows user)
