# skl-go  

## Features  
* An enterprise api for web applications.  
* Written in beego  
* Including user,usergroup,orginazation,role and priviledge management  

## Environment Support  
* beego 1.9.0  
* golang go1.9.2  

## Installation  
* install golang by yourself  
* install beego by yourself  
* go install skl-golang
